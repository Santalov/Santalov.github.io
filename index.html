Инструкция по развертке на localhost

1. клонируем репозиторий https://github.com/Santalov/Santalov.github.io

2. устанавливаем nginx
sudo apt-get install nginx

3. 

server {
    listen {порт} default_server;
    listen [::]:{порт} default_server;
    server_name _;
    location / {
        root {папка, куда склонирован репозиторий};
   }
}
Это надо засунуть фаил /etc/nginx/conf.d/название.conf
Возможно понадобится раскомментить это include после утсановки: include /etc/nginx/conf.d/*.conf; Порт лучше выбрать такой, чтобы не конфликтовал с http по умолчанию и другими решениями.

4. Заходим на http://localhost:{порт}/local или на http://localhost:{порт}/emscripten-local

Инструкция по развертке на любом домене

1. Заливаем на сервак содержимое папки /demo (или другой папки, в зависимости от того, что нужно)

2. Меняем в файлах worker.js, payment-manifest.json, /checkout/checkout.js эндпоинты на нужный домен (файла /checkout/checkout.js может не быть)
